What is the difference between GitHub and Git

Git is the bow, your code is the arrow, and Github is the target.

Simply, git is a software, it is also a local “warehouse” at your pc where you can save different versions of the code and documents. GitHub is an online “warehouse”, where you can also save your code and its version.
